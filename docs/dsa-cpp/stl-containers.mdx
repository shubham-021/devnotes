## Pairs

A `pair` holds two values of potentially different types.

```cpp
pair<int, int> p = {1, 3};
cout << p.first << " " << p.second; // 1 3

// Nested pairs
pair<int, pair<int, int>> p = {1, {2, 4}};
cout << p.first;         // 1
cout << p.second.first;  // 2
cout << p.second.second; // 4

// Array of pairs
pair<int, int> arr[] = {{1,2}, {2,3}, {4,5}};
```

## Vectors

Dynamic arrays that can grow and shrink.

### Basic Operations

```cpp
vector<int> v;

v.push_back(1);   // Copies value into vector
v.emplace_back(2); // Constructs in-place (slightly faster)

// Vector of pairs
vector<pair<int,int>> vec;
vec.push_back({1, 2});    // Needs braces
vec.emplace_back(1, 2);   // No braces needed
```

### Initialization

```cpp
vector<int> v(5, 100);  // {100, 100, 100, 100, 100}
vector<int> v(5);       // {0, 0, 0, 0, 0} - default initialized
vector<int> v2(v1);     // Copy of v1
```

### Iterators

Iterators are pointers to container elements.

```cpp
vector<int>::iterator it = v.begin(); // Points to first element
it++;
cout << *it; // Dereference to get value

// Iterator types
v.begin();  // First element
v.end();    // One past last element
v.rbegin(); // Last element (reverse)
v.rend();   // One before first (reverse)
```

### Accessing Elements

```cpp
cout << v[0];      // Direct access (no bounds checking)
cout << v.at(0);   // Bounds-checked access
cout << v.back();  // Last element
cout << v.front(); // First element
```

### Looping

```cpp
// Iterator loop
for (auto it = v.begin(); it != v.end(); it++) {
    cout << *it << " ";
}

// Range-based loop (cleanest)
for (auto x : v) {
    cout << x << " ";
}
```

### Modifying

```cpp
// Erase
vector<int> v = {10, 20, 30, 40};
v.erase(v.begin() + 1);              // {10, 30, 40}

vector<int> v = {10, 20, 30, 40, 50};
v.erase(v.begin() + 1, v.begin() + 3); // {10, 40, 50} - range is [start, end)

// Insert
vector<int> v(2, 100);              // {100, 100}
v.insert(v.begin(), 300);           // {300, 100, 100}
v.insert(v.begin() + 1, 2, 10);     // {300, 10, 10, 100, 100}

// Insert from another vector
vector<int> copy(2, 50);
v.insert(v.begin(), copy.begin(), copy.end());
```

### Utility Functions

```cpp
v.size();      // Number of elements
v.pop_back();  // Remove last element
v.clear();     // Remove all elements
v.empty();     // Returns true if empty

// Swap contents
v1.swap(v2);   // v1 gets v2's elements and vice versa
```

## List

Doubly-linked list with O(1) insertion/deletion at both ends.

```cpp
list<int> ls;

ls.push_back(2);     // {2}
ls.emplace_back(4);  // {2, 4}
ls.push_front(5);    // {5, 2, 4}
ls.emplace_front(6); // {6, 5, 2, 4}
```

Unlike vectors, lists allow efficient front insertion but don't support random access (`ls[0]` is invalid).

## Deque

Double-ended queue â€” combines vector and list benefits.

```cpp
deque<int> dq;

dq.push_back(1);     // {1}
dq.emplace_back(2);  // {1, 2}
dq.push_front(4);    // {4, 1, 2}
dq.emplace_front(3); // {3, 4, 1, 2}

dq.pop_back();       // {3, 4, 1}
dq.pop_front();      // {4, 1}

cout << dq.front();  // 4
cout << dq.back();   // 1
```

Supports random access like vectors, plus O(1) front operations like lists.

## Stack

Last-In-First-Out (LIFO) container.

```cpp
stack<int> st;

st.push(1);     // {1}
st.push(2);     // {2, 1}
st.push(3);     // {3, 2, 1}
st.emplace(5);  // {5, 3, 2, 1}

cout << st.top();   // 5 (no st[0] - only top access)
st.pop();           // {3, 2, 1}
cout << st.size();  // 3
cout << st.empty(); // false
```

## Queue

First-In-First-Out (FIFO) container.

```cpp
queue<int> q;

q.push(1);  // {1}
q.push(2);  // {1, 2}
q.push(3);  // {1, 2, 3}

cout << q.front(); // 1 (first in)
cout << q.back();  // 3 (last in)

q.back() += 5;     // Can modify back
cout << q.back();  // 8

q.pop();           // {2, 3, 8} - removes front
cout << q.front(); // 2
```

## Priority Queue

A heap that keeps the largest (or smallest) element at the top.

### Max Heap (Default)

```cpp
priority_queue<int> pq;

pq.push(5);     // {5}
pq.push(2);     // {5, 2}
pq.push(8);     // {8, 5, 2}
pq.emplace(10); // {10, 8, 5, 2}

cout << pq.top(); // 10 (maximum)
pq.pop();         // {8, 5, 2}
```

### Min Heap

```cpp
priority_queue<int, vector<int>, greater<int>> pq;

pq.push(5);     // {5}
pq.push(2);     // {2, 5}
pq.push(8);     // {2, 5, 8}

cout << pq.top(); // 2 (minimum)
```

### Time Complexity

| Operation | Complexity |
|-----------|------------|
| `push` | O(log n) |
| `pop` | O(log n) |
| `top` | O(1) |
